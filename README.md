# Portfolio Website Home Page

A modern portfolio website built with Next.js and React. The original project is available at https://www.figma.com/design/2vOUMz9HJJPBLwr3r6l65r/Portfolio-Website-Home-Page.

## Tech Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Radix UI Components

## Running the code

Run `npm install` to install the dependencies.

Run `npm run dev` to start the development server.

The application will be available at http://localhost:3000.

## Building for Production

Run `npm run build` to create a production build.

Run `npm start` to run the production server.
